# TAU-R1: Visual Language Model for Traffic Anomaly Understanding

**TAU-R1** is a two-layer visual language model based framework designed for traffic anomaly understanding, focusing on real-world roundabout scenarios. 

## Dataset
The dataset for this project, **Roundabout-TAU**, is available on [Hugging Face](https://huggingface.co/datasets/yl4300/Roundabout-TAU/tree/main).
